# @xifan-coder/mem

Cross-session memory persistence and retrieval for XiFanCoder.

## Features

- SQLite-backed observation storage
- Full-text search across sessions
- Privacy-aware content filtering
- Queue-based async observation processing
- Timeline and search MCP tools

## Installation

```bash
npm install @xifan-coder/mem
```
