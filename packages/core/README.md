# @xifan-coder/core

Core package for XiFanCoder - Agent Loop, LLM Drivers, Built-in Tools, and MCP Server.

## Features

- Agent loop with configurable tool dispatching
- Multi-provider LLM drivers (Anthropic, OpenAI, LiteLLM, Ollama)
- Built-in tools (bash, file read/write, search, LSP)
- MCP server with TLS and connection limiting
- Session and database management
- Crash reporting with sensitive data redaction

## Installation

```bash
npm install @xifan-coder/core
```
