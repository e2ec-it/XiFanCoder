# @xifan-coder/cli

Command-line interface for XiFanCoder - an AI Agent CLI Coding Tool.

## Installation

```bash
npm install -g @xifan-coder/cli
```

## Usage

```bash
# Start interactive REPL
xifan-coder

# Show help
xifan-coder help

# Initialize project config
xifan-coder init --config
```

See the [main README](https://github.com/e2ec-it/XiFanCoder#readme) for full documentation.
