# @xifan-coder/plugin-bus

Plugin discovery, registry, and lifecycle management for XiFanCoder.

## Features

- Automatic plugin discovery from global, project, and node_modules paths
- Plugin manifest validation
- Stdio-based plugin process management
- Permission-level enforcement (L0-L3)

## Installation

```bash
npm install @xifan-coder/plugin-bus
```
